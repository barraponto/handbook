Welcome!

This is a rough draft of the handbook for contributing to Backdrop.

Contributing

Work is just beginning, so if you have thoughts or ideas in general on the handbook, please submit an issue on GitHub.

New Pages

New pages can be submitted with a pull request. Simply fork this repository, add a new page to the _pages directory, and submit a pull request. You can also navigate to the _pages directory within this repository and either edit a file or create a new one which will create a pull request for you.

Organization

Organization of the handbook is in progress. Currently there are 3 main sections:

  • About
  • Developers
  • Users Guide
If you have thoughts, please submit an issue.
